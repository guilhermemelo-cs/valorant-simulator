## WHAT MY SYSTEM DO?

Until now, it works like a simulator of matches of valorant. Basically, you can start and play for 13 rounds. Before the round starts, you can buy some guns in the shop. After this, the system decide if you win or you loss. The messages in the print is in PT-BR. After the final round, the system show if you win the match or not.
